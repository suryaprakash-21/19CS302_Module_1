# EX 2 C program to check whether the attendance is PRESENT using simple if statement.
## DATE:08/06/2026
## AIM:
To write a program to check whether the attendance is PRESENT using simple if statement.

## Algorithm
1. Start the program.
2. Read the attendance status from the user (e.g., 'P' for Present).
3. Use a simple if statement to check whether the attendance status is 'P'.
4. If the condition is true, display "Student is PRESENT".
5. Stop the program. 
  

## Program:
```
#include<stdio.h>
int main()
{
    char c;
    scanf("%c",&c);
    if(c=='A'){
        printf("Absent");
    }
    return 0;
}
```
